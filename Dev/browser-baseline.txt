# Google Chrome (User-level)
# -------------------------

# Allow extensions only from Chrome Web Store (Manifest V2 only)
# LGPO: User Configuration\Administrative Templates\Google\Google Chrome\Extensions\Configure extension installation whitelist
HKCU\Software\Policies\Google\Chrome\ExtensionInstallAllowlist|1|REG_SZ|*

# Enable password manager
# LGPO: User Configuration\Administrative Templates\Google\Google Chrome\Password manager enabled
HKCU\Software\Policies\Google\Chrome|PasswordManagerEnabled|REG_DWORD|1

# Disable Developer Tools
# LGPO: User Configuration\Administrative Templates\Google\Google Chrome\Developer Tools availability
HKCU\Software\Policies\Google\Chrome|DeveloperToolsAvailability|REG_DWORD|2

# Enable Safe Browsing
# LGPO: User Configuration\Administrative Templates\Google\Google Chrome\Enable Safe Browsing
HKCU\Software\Policies\Google\Chrome|SafeBrowsingEnabled|REG_DWORD|1
